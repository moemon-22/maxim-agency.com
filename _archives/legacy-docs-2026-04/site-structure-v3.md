### Section 0: Hero（ファーストビュー）

**方向性**: ブランドインパクト型 — ロゴを大きく見せ、スクロールで全容が明らかになる

**構成**:
```
[ファーストビュー: 100vh フルスクリーン]

中央: MAXIMブランドロゴ（カスタムジオメトリックフォント）
      "MAXIM" + ブルーアクセントライン + "AGENCY"
      ※ ロゴ画像を使用（テキストではない）

ロゴ下（テキスト）: We carry AX into your business.
      ※ タグライン。ロゴ画像の外にテキストとして配置

その下: 中小企業のためのAI Transformation パートナー。

画面下部: スクロールインジケーター（↓ or アニメーション付き矢印）
```

**スクロール後に現れるコンテンツ（Section 0.5的な位置）**:
```
キャッチコピー（H1）:
AIの導入、
何から始めればいい？

——その答えを、一緒に見つけます。

サブコピー:
業務分析からツール選定・導入・定着まで。
提案で終わらない、構築と引き渡しの伴走型支援。

CTAボタン: 無料相談を予約する
補足: オンライン / 30分 / 無料
```

**デザインノート**:
- ファーストビュー: ティール〜ダークのグラデーション背景、100vh
- ロゴは画面中央に大きく配置（幅40〜50%程度）
- サブコピー1行はロゴ直下に控えめに（白テキスト、opacity低め）
- スクロールインジケーターは画面下部中央にアニメーション付き
- スクロール後のキャッチコピーセクションはフェードイン
- CTAボタンはミニマルなアウトライン or ソリッド白

### Section 1: Pain Points（課題提起）

**セクションラベル**: CHALLENGES
**タイトル**: こんな課題、抱えていませんか？

**項目（4つ — シンプルに絞る）**:

```
1. AIを使いたいが、何から手をつければいいかわからない
   → ツールが多すぎて選べない。導入しても定着しない。

2. 業務のデジタル化が進まず、手作業が残っている
   → Excel転記、紙の管理、属人化した業務フロー。

3. DXを外注したいが、相談先がわからない
   → 大手コンサルは高すぎる。IT企業は技術寄りで経営がわからない。

4. 導入しても、社内で使いこなせない
   → ツールを入れただけで終わり。研修も引き継ぎもなかった。
```

**デザインノート**:
- 4枚のカード、2×2グリッド（PC） / 1列（SP）
- ミニマルなラインアイコン + タイトル + 1行説明

### Section 2: Solution（解決策 — MAXIMの強み）

**セクションラベル**: OUR APPROACH
**タイトル**: コンサルではない。実行者です。

**リード文**:
```
MAXIM AGENCYは、提案書を出して終わるコンサルではありません。
あなたの事業の中にAXの仕組みを構築し、
チームが自走できる状態にして引き渡す——それが私たちの仕事です。

大手広告代理店でのマーケティング経験と、
建設会社での経営・DX推進の実体験を持つコンサルタントが、
経営者目線で伴走します。
```

**3つの柱（カード）**:

```
01  診断 — 経営課題から逆算する
    現状の業務フローを丁寧にヒアリング。
    技術ありきではなく、経営課題から逆算して
    「何を・どこから・どう変えるか」を一緒に設計します。

02  構築 — 提案で終わらない
    ツール選定・セットアップ・業務フローの再構築まで実行。
    「入れて終わり」ではなく、あなたの事業の中に仕組みを作ります。

03  引き渡し — 依存ではなく、自走へ
    経営者・スタッフへの研修とマニュアル納品。
    チームが自走できる状態を確認してから、次のステップへ。
    回を重ねるごとに、あなたの組織の自走力が上がります。
```

### Section 4: Process（導入の流れ）

**セクションラベル**: PROCESS
**タイトル**: 進め方

**導入の入口**:
```
まずは無料相談（30分・オンライン）から。
現在の課題やお悩みをヒアリングし、AI/DXで改善できるポイントを整理します。
ここから先は、マイルストーンごとに3つのフェーズを繰り返します。
```

**3フェーズ（マイルストーン型）**:

```
Phase 1: 診断
業務フローの分析、AI/DXで改善可能なポイントの特定。
明確なKPIを設定し、ゴールを合意します。

Phase 2: 構築
ツール選定・導入設計・業務フローの再構築・テスト運用。
途中経過を随時共有しながら、仕組みを作り上げます。

Phase 3: 引き渡し
経営者・スタッフへの操作研修、マニュアル納品、自走確認。
チームが自走できる状態を確認してから完了。
→ 次のマイルストーンへ。
```

**補足テキスト**:
```
この3フェーズを1つのマイルストーンとして完了させ、
次の課題へ進みます。回を重ねるごとに、
あなたの組織のAX力が上がっていきます。
```

**デザインノート**:
- 3つのフェーズを横並びカード（PC） / 縦並び（SP）
- フェーズ間を矢印でつなぐ循環型のビジュアル（→ → → 次のマイルストーンへ）
- 最初に「無料相談」をエントリーポイントとして明示
- 最後のフェーズに循環を示すアイコン（リピート矢印等）
